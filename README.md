### Запуск через Docker

Протестировать решение можно через Docker введя в терминале команду (Заменив вместо **** любой свободный порт, например 8000):

```
docker run -dp ****:8000 sidikayfuy/stripe-django-testwork:latest & start http://localhost:****/
```
